> Wrapper around New Relic&#39;s Synthetics API

## Installation

```sh
$ npm install --save new-relic-synthetics
```

## Usage

```js
var newRelicSynthetics = require('new-relic-synthetics');

newRelicSynthetics('Rainbow');
```
